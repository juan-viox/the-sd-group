05 · Hero Concept

Not a property shot. A presence.

Concept — Curtain Reveal · Golden Hour

A Ridgewood Tudor at 5:47pm. The camera doesn't move. The light does.

A slow, cinematic 4K curtain-reveal of a Ridgewood Tudor or Colonial at golden hour. The frame holds — stone steps, leaded glass, a hedge just turning in late afternoon light. As the viewer scrolls, theater-curtain panels part and Sarah's name dissolves in alongside a single editorial tagline. No search bar. No stock drone. No swelling music. No "Search Now" button competing for attention. Just the house, the light, and the quiet assertion that this is where the brand lives.

Generated with Kling 3.0 4K + Seedance 2.0 in parallel — user picks the winner. Frames extracted at 24fps, scroll-scrubbed via GSAP ScrollTrigger. First frame preloaded separately for instant display on load; remaining frames stream in.

"Bergen County, brought home."

06 · AI Voice Concierge

A bilingual agent that answers the phone at 11:47pm.

Built on ElevenLabs ConvAI with a Twilio phone line and full bilingual knowledge base. Handles qualification, neighborhood Q&A, scheduling, and after-hours overflow — in English or Spanish, automatic language detection.

i
Language detection + switching — caller's first language wins. Mid-call code-switching supported.
ii
Bergen neighborhood depth — school districts, median prices, commute data, current inventory counts for every town in the service area.
iii
The SD Method walkthrough — can narrate all 12 steps of the listing process on demand, in either language.
iv
Qualification + handoff — captures budget, timeline, property type, then books a call on Sarah's calendar or emails a warm intro.
v
Guardrails — prompt-injection defense, profanity filter, graceful boundary on topics outside real estate.

07 · Technical Approach

Built to be owned, not rented.

No templates. No WordPress bloat. No brokerage-subdomain lock-in. A single deployable HTML artifact with a complete design system exported for client handoff.

Stack

Vanilla + GSAP

Single index.html, inline CSS + JS. GSAP 3.12.5 + ScrollTrigger from CDN. Zero build step. Zero framework dependency. Loads in under 1.5s.

Hosting

Vercel + GitHub

Push to main → auto-deploy. Custom domain at thesdgroup.com (pending availability check). Edge caching, free SSL, 100 Lighthouse.

Media

Kling 3.0 + Seedance 2.0

Native 4K video via Kling, Apple-keynote motion via Seedance. Run in parallel — the user picks the winner frame-for-frame.

Voice

ElevenLabs + Twilio

Bilingual ConvAI agent with a dedicated North NJ phone number. RAG over the full brand knowledge base. 15-minute max call duration.

SEO

Schema.org + OG + Sitemap

RealEstateAgent markup, Open Graph cards, Twitter summary cards, XML sitemap, neighborhood long-tail pages, full mobile-first indexing.

Design System

Portable DESIGN.md

Full token export — colors, type scale, spacing, motion — in Stitch-compatible format. Ready for flyers, Instagram templates, email signature.

09 · What We Won't Do

Twelve patterns we will not ship.

The design rules that keep the site out of generic-realtor territory. These are enforced across every phase.

Stock drone shots of suburban cul-de-sacs
Corporate blue + white KW-default palette
Lorem Ipsum or AI-generated testimonials
Auto-playing music or voice on page load
Search bar as primary hero element
Inter or generic Google sans for display
Purple / violet gradients + neon accents
Carousel-of-everything homepage
MLS-iframe grids with inconsistent photo quality
Red CTA buttons (KW parent-brand collision)
Script fonts signaling "feminine luxury"
Animated stat counters with no context
